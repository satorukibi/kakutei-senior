# 申告準備ノート（高齢者向け）

スマホで使える、確定申告の**準備用メモ**アプリです。  
医療費・経費・収入（年金など）を記録し、申告前に数字を整理できます。

**※ 税務相談の代わりにはなりません。試用版です。**

## 公開URL

https://satorukibi.github.io/kakutei-senior/

## 機能

- **医療費** … 病院・薬局の支出を記録
- **経費** … 保険料・寄付・仕事の経費など
- **収入** … 給料・賃金、アルバイト、国民年金・厚生年金・年金3〜6など
- **まとめ** … 年間の集計（申告準備用）
- **準備** … 必要書類のチェックリスト

## 使い方

1. 上のURLをスマホのブラウザで開く
2. **インストール** または **ホーム画面に追加**
3. 日々の支出・収入を記録
4. 申告シーズン（2〜3月）に「まとめ」「準備」を確認

## 高齢者・家族向けの配慮

- 文字を大きく、ボタンを押しやすく設計
- 年金は種類を調べなくてよい（国民年金・厚生年金・年金3〜6）
- データはスマホ内に保存（次回も続きから）

## 注意

- 正式な申告数字は、税務署の書類・控除証明書を優先してください
- 分からないことは税務署（**#3110**）や家族に相談してください

## 技術

HTML / CSS / JavaScript（PWA）。サーバー不要で動作します。
